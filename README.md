# 5-Axis CNC Controller
This is a Python-based GUI application for controlling a 5-axis CNC machine over USB.

# Requirements
Python 3.8+
Libraries listed in requirements.txt
Installation
Install Python
Install dependencies: pip install -r requirements.txt
Run the application: python main.py
# Usage
Use sliders to move each axis
Click 'Start' to begin operation
Click 'Stop' to halt operation
# Note
This is a basic implementation. You'll need to adapt the USB communication code to match your specific microcontroller and CNC setup.
